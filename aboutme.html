I used Python to create a filter on an image by changing the color value of the pixels to either dark blue, red, light blue, or yellow, depending on the intensity or shade of the pixel in the original image. The filtered image is colored in a way similar to that of the Obamicon image. I first used a "for" loop to find the sum of the RGB values in each pixel, and then I used "if," "elif," and "else" statements to change the original color to a certain color depending on the sum of the color values in the pixel.
